## 💻 CODE EXAMPLES

### Multi-Platform Scraping (ALL IN ONE)

```python
from scrapeflux import (
    GoogleExtractor,
    MapsExtractor,
    TwitterExtractor,
    MultiIPClient
)

# ONE setup for ALL platforms
client = MultiIPClient(proxies)

# Extract from multiple sources - SIMPLE!
search_results = await GoogleExtractor(client).search("restaurants NYC", 100)
map_places = await MapsExtractor(client).search_places("restaurants", "40.7128,-74.0060", "5km")
tweets = await TwitterExtractor(client).get_tweets("#NYC", 50)

# VS Others: You need 3+ different libraries, 1000+ lines of code, and still get blocked
```

### AI-Powered Extraction

```python
from scrapeflux import LLMEngine, AIAgent

# Generate schema with AI - AUTO
engine = LLMEngine()
schema = await engine.generate_schema("Extract products with prices")

# Autonomous agent - Just describe what you want!
agent = AIAgent()
result = await agent.execute("Scrape all job listings for engineers")
```
